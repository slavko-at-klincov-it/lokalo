★ Final Onboarding Flow

Zwei Beats

Einmaliger Onboarding-Flow auf erstem App-Start. Beat 1 zeigt das Markenversprechen, Beat 2 die personalisierbaren Einstellungen. Kein Auto-Download — der User wählt sein erstes Modell aktiv aus der Bibliothek.

1Sternenwort + Versprechen
~5 s · Tap/Swipe → Beat 2

Konstellation formt sich, Lokalo emerged im Zentrum, darunter die vier Promises („Läuft auf dem Gerät. Kein Lokalo-Backend. Keine Werbung. Keine Telemetrie.") gestapelt. Unten: pulsierender Hint „Zum Starten wischen →". Click/Tap/Swipe-left → slide-out.

2Personalisieren
~3 s reveal · 4 Optionen + Loslegen

Header „Personalisieren · kann später jederzeit geändert werden". Vier Karten mit Toggles: Mikrofon · Benachrichtigungen · Modelle ohne WLAN · Standard-Modell (Qwen 2.5 0.5B). Loslegen-Button schließt das Onboarding.

1Atemzug
Apple Hello · Pure Restraint

Atmender Punkt → Satz „Alles bleibt hier." → Fade. Anti-Onboarding. Vertraut dem User dass er das Offensichtliche selbst kapiert.

2Konstellation
Star Walk · Brilliant Night

Punktwolke fliegt von außen rein, formt ein Gehirn-Blob mit Verbindungen. Visualisiert das LLM literal als „etwas das auf deinem iPhone wohnt".

3Brief von daheim
Wes Anderson · Pen Pals

Handgeschriebener Brief slidet rein, Text tippt sich Wort für Wort. Tactile, warm, persönlich. Analog gegen Hightech.

4Tokens
Cyberpunk Monospace · Linear Poster

Token-Capsules streamen vom Bildschirmboden hoch, „Lokalo" emerged aus dem Stream. Meta: das was die App tut IST das Welcome.

5Lichtschalter
Apple Keynote · Quiet Luxury

Drei Worte erscheinen einzeln, mergen zu „Lokalo", Tap → Weißblitz → Chat. Drei Werte → ein Name. Cinematic.
